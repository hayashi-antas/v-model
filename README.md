# V字モデル

V字モデルの各工程と成果物を理解するためのシンプルなWebサイトです。トップページから要求分析～受入テストまでのフェーズ解説や、要件定義書・設計書・テスト資料などの具体的なドキュメントサンプルへアクセスできます。静的HTMLとCSS/JSのみで構成しているため、`index.html` をブラウザで開くだけで閲覧できます。

## ディレクトリ概要
- `index.html` : V字モデルのスライドと成果物リンク集
- `phases/` : 10工程それぞれの目的と活動内容
- `docs/` : 成果物の概要と活用ポイント
- `assets/` / `styles.css` : 波アニメーションと共通スタイル
